🛠️ 本地预览与构建

本套文档支持使用 mdBook 编译成静态网页。

1. 安装 mdBook

# 使用 Cargo 安装
cargo install mdbook

# 安装 Mermaid 预处理器（用于渲染架构图）
cargo install mdbook-mermaid

2. 本地运行

# 启动本地开发服务器，支持热重载
mdbook serve

启动后，可在浏览器中访问 http://localhost:3000 进行实时预览。

3. 编译静态网页

# 编译输出静态网页到 book/ 目录
mdbook build
