💾 关系型数据库与高性能缓存​

在海量高吞吐系统中，数据存储与一致性是系统最脆弱也是最核心的地方。

  • MySQL 关系型数据库：
    • InnoDB 索引物理结构与 B+ 树路由
    • InnoDB MVCC 多版本链与 Read View 隔离算法
    • 物理回滚日志 (Undo) 与主从复制机制
    • MySQL 慢查询调优与高性能开发规约
  • Redis 高性能缓存：
    • 自研多路复用 I/O 线程模型与核心数据结构
    • Redis 哨兵与 Cluster 集群高可用构筑
    • 缓存与 MySQL 数据双写强一致性方案
    • 击穿、穿透、雪崩工业级防御 (Redisson DCL / 布隆过滤器)

🌐 分布式系统核心底盘​

探究跨物理机器如何保持协同、数据一致及网络容错。

  • 一致性协议：Paxos 与 Raft 强一致性共识算法
  • 分布式锁：基于 ZooKeeper / Curator 临时顺序节点的排他锁实现
  • 分布式事务：2PC, 3PC 及 TCC 补偿事务模型底层设计
  • 消息队列：高吞吐 RocketMQ / Kafka 零拷贝与 ACK 可靠性模型

🦀 Rust 极速探索​

进入无 GC、具备编译期内存安全的现代化系统级编程语言领域。

  • 所有权与生命周期：所有权借用检查器与编译期生命周期约束
  • 内存模型：RAII、裸指针、RefCell 与智能指针底层物理内存
  • 无畏并发：Send 与 Sync 约束下的多线程无显示锁高并发安全
