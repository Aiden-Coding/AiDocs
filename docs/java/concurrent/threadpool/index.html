зәҝзЁӢжұ ж·ұеәҰи§ЈжһҗдёҺи°ғдјҳвҖӢ

еңЁ Java й«ҳе№¶еҸ‘еә”з”ЁдёӯпјҢзәҝзЁӢжұ жҳҜз®ЎзҗҶе’ҢеӨҚз”ЁзәҝзЁӢзҡ„ж ёеҝғеҲ©еҷЁгҖӮеҗҲзҗҶй…ҚзҪ®зәҝзЁӢжұ еҸӮж•°гҖҒзҗҶи§Је…¶е·ҘдҪңеҺҹзҗҶд»ҘеҸҠжҺҢжҸЎеҠЁжҖҒи°ғдјҳжүӢж®өпјҢжҳҜй«ҳзә§ Java е·ҘзЁӢеёҲзҡ„еҝ…еӨҮжҠҖиғҪгҖӮ

дёҖгҖҒ зәҝзЁӢжұ ж ёеҝғеҸӮж•°дёҺе·ҘдҪңжөҒзЁӢвҖӢ

Java дёӯзҡ„зәҝзЁӢжұ ж ёеҝғе®һзҺ°зұ»жҳҜ ThreadPoolExecutorгҖӮ

1. ж ёеҝғеҸӮж•°пјҲ7 еӨ§еҸӮж•°пјүвҖӢ

public ThreadPoolExecutor(
    int corePoolSize,               // ж ёеҝғзәҝзЁӢж•°
    int maximumPoolSize,            // жңҖеӨ§зәҝзЁӢж•°
    long keepAliveTime,             // йқһж ёеҝғз©әй—ІзәҝзЁӢеӯҳжҙ»ж—¶й—ҙ
    TimeUnit unit,                  // еӯҳжҙ»ж—¶й—ҙеҚ•дҪҚ
    BlockingQueue workQueue, // д»»еҠЎйҳҹеҲ—
    ThreadFactory threadFactory,    // зәҝзЁӢе·ҘеҺӮ
    RejectedExecutionHandler handler // жӢ’з»қзӯ–з•Ҙ
) {
    // ...
}
  1. corePoolSizeпјҡзәҝзЁӢжұ дёӯеёёй©»зҡ„ж ёеҝғзәҝзЁӢж•°гҖӮеҚідҪҝиҝҷдәӣзәҝзЁӢеӨ„дәҺз©әй—ІзҠ¶жҖҒпјҢд№ҹдёҚдјҡиў«й”ҖжҜҒпјҲйҷӨйқһи®ҫзҪ®дәҶ allowCoreThreadTimeOutпјүгҖӮ
  2. maximumPoolSizeпјҡзәҝзЁӢжұ е…Ғи®ёеҲӣе»әзҡ„жңҖеӨ§зәҝзЁӢж•°гҖӮеҪ“йҳҹеҲ—ж»Ўдё”е·ІеҲӣе»әзәҝзЁӢж•°е°ҸдәҺжңҖеӨ§зәҝзЁӢж•°ж—¶пјҢдјҡз»§з»ӯеҲӣе»әж–°зәҝзЁӢгҖӮ
  3. keepAliveTimeпјҡйқһж ёеҝғзәҝзЁӢй—ІзҪ®ж—¶зҡ„еӯҳжҙ»ж—¶й—ҙгҖӮеҪ“зәҝзЁӢжұ дёӯзҡ„зәҝзЁӢж•°еӨ§дәҺ corePoolSize ж—¶пјҢеӨҡдҪҷзҡ„з©әй—ІзәҝзЁӢеңЁеӯҳжҙ»ж—¶й—ҙиҫҫеҲ°еҗҺдјҡиў«й”ҖжҜҒгҖӮ
  4. unitпјҡkeepAliveTime зҡ„ж—¶й—ҙеҚ•дҪҚгҖӮ
  5. workQueueпјҡеӯҳж”ҫеҫ…жү§иЎҢд»»еҠЎзҡ„йҳ»еЎһйҳҹеҲ—гҖӮ
  6. threadFactoryпјҡз”ЁдәҺеҲӣе»әж–°зәҝзЁӢзҡ„е·ҘеҺӮпјҢеҸҜд»ҘиҮӘе®ҡд№үзәҝзЁӢеҗҚз§°гҖҒе®ҲжҠӨиҝӣзЁӢеұһжҖ§зӯүгҖӮ
  7. handlerпјҡеҪ“зәҝзЁӢжұ е’ҢйҳҹеҲ—йғҪж»ЎдәҶд№ӢеҗҺзҡ„д»»еҠЎжӢ’з»қзӯ–з•ҘгҖӮ

2. зәҝзЁӢжұ е·ҘдҪңжөҒзЁӢвҖӢ

жіЁж„ҸпјҡзәҝзЁӢжұ еңЁеҲҡеҲӣе»әж—¶пјҢеҶ…йғЁй»ҳи®ӨжҳҜжІЎжңүзәҝзЁӢзҡ„гҖӮеҸӘжңүеҪ“д»»еҠЎжҸҗдәӨиҝӣжқҘж—¶пјҢжүҚдјҡејҖе§ӢеҲӣе»әзәҝзЁӢгҖӮеҰӮжһңжғіжҸҗеүҚеҲӣе»әж ёеҝғзәҝзЁӢпјҢеҸҜд»Ҙи°ғз”Ё prestartAllCoreThreads()гҖӮ

дәҢгҖҒ йҳ»еЎһйҳҹеҲ—дёҺжӢ’з»қзӯ–з•ҘвҖӢ

1. еёёи§Ғйҳ»еЎһйҳҹеҲ—пјҲBlockingQueueпјүвҖӢ

  • ArrayBlockingQueueпјҡеҹәдәҺж•°з»„зҡ„жңүз•Ңйҳ»еЎһйҳҹеҲ—пјҢжҢү FIFO жҺ’еәҸгҖӮеҝ…йЎ»жҢҮе®ҡеӨ§е°ҸгҖӮ
  • LinkedBlockingQueueпјҡеҹәдәҺй“ҫиЎЁзҡ„йҳ»еЎһйҳҹеҲ—пјҢжҢү FIFO жҺ’еәҸгҖӮй»ҳи®ӨеӨ§е°Ҹдёә Integer.MAX_VALUEпјҲж— з•ҢпјүпјҢеңЁй«ҳе№¶еҸ‘дёӢжһҒжҳ“еҜјиҮҙ OOMпјҢеӣ жӯӨеҝ…йЎ»жүӢеҠЁжҢҮе®ҡе®№йҮҸгҖӮ
  • SynchronousQueueпјҡдёҚеӯҳеӮЁе…ғзҙ зҡ„йҳ»еЎһйҳҹеҲ—гҖӮжҜҸдёӘжҸ’е…Ҙж“ҚдҪңеҝ…йЎ»зӯүеҫ…еҸҰдёҖдёӘзәҝзЁӢзҡ„з§»йҷӨж“ҚдҪңпјҢеҗһеҗҗйҮҸжһҒй«ҳгҖӮExecutors.newCachedThreadPool() дҪҝз”Ёзҡ„е°ұжҳҜе®ғгҖӮ
  • PriorityBlockingQueueпјҡж”ҜжҢҒдјҳе…Ҳзә§жҺ’еәҸзҡ„ж— з•Ңйҳ»еЎһйҳҹеҲ—гҖӮ

2. еҶ…зҪ®жӢ’з»қзӯ–з•ҘпјҲRejectedExecutionHandlerпјүвҖӢ

  • AbortPolicyпјҲй»ҳи®ӨпјүпјҡзӣҙжҺҘжҠӣеҮә RejectedExecutionException ејӮеёёпјҢйҳ»жӯўзі»з»ҹжӯЈеёёиҝҗиЎҢгҖӮ
  • CallerRunsPolicyпјҡвҖңи°ғз”ЁиҖ…иҝҗиЎҢвҖқзӯ–з•ҘгҖӮиҜҘзӯ–з•Ҙж—ўдёҚдјҡжҠӣејғд»»еҠЎпјҢд№ҹдёҚдјҡжҠӣеҮәејӮеёёпјҢиҖҢжҳҜе°Ҷд»»еҠЎеӣһйҖҖз»ҷжҸҗдәӨд»»еҠЎзҡ„зәҝзЁӢпјҲеҰӮдё»зәҝзЁӢпјүеҺ»жү§иЎҢгҖӮ
  • DiscardOldestPolicyпјҡдёўејғйҳҹеҲ—дёӯзӯүеҫ…жңҖд№…зҡ„д»»еҠЎпјҢз„¶еҗҺйҮҚж–°е°қиҜ•жҸҗдәӨеҪ“еүҚд»»еҠЎгҖӮ
  • DiscardPolicyпјҡзӣҙжҺҘдёўејғд»»еҠЎпјҢдёҚдәҲд»»дҪ•еӨ„зҗҶд№ҹдёҚжҠӣеҮәејӮеёёгҖӮ

дёүгҖҒ зәҝзЁӢжұ еӨ§е°ҸеҰӮдҪ•еҗҲзҗҶй…ҚзҪ®пјҹвҖӢ

зәҝзЁӢжұ еӨ§е°Ҹй…ҚзҪ®дёҚеҪ“пјҢиҰҒд№ҲеҜјиҮҙ CPU йў‘з№ҒеҲҮжҚўдёҠдёӢж–ҮпјҲзәҝзЁӢж•°иҝҮеӨҡпјүпјҢиҰҒд№ҲеҜјиҮҙзі»з»ҹиө„жәҗж— жі•е……еҲҶеҲ©з”ЁпјҲзәҝзЁӢж•°иҝҮе°‘пјүгҖӮ

1. з»Ҹе…ёдј°з®—е…¬ејҸвҖӢ

ж №жҚ®д»»еҠЎзҡ„еұһжҖ§пјҢеҸҜд»Ҙе°Ҷд»»еҠЎеҲҶдёә CPU еҜҶйӣҶеһӢ е’Ң IO еҜҶйӣҶеһӢпјҡ

CPU еҜҶйӣҶеһӢпјҡ

  • зү№зӮ№пјҡд»»еҠЎдё»иҰҒж¶ҲиҖ— CPU иө„жәҗпјҢеҮ д№ҺжІЎжңүйҳ»еЎһгҖӮ
  • й…ҚзҪ®е…¬ејҸпјҡзәҝзЁӢж•° = CPU ж ёеҝғж•° + 1
  • еҺҹзҗҶпјҡйўқеӨ–еӨҡй…ҚзҪ®дёҖдёӘзәҝзЁӢжҳҜдёәдәҶйҳІжӯўзәҝзЁӢеӣ дёәеҒ¶еҸ‘зҡ„йЎөзјәеӨұгҖҒзәҝзЁӢжҡӮеҒңзӯүеҺҹеӣ еҜјиҮҙ CPU з©әй—ІпјҢдҝқиҜҒ CPU зҡ„й«ҳеҲ©з”ЁзҺҮгҖӮ

IO еҜҶйӣҶеһӢпјҡ

  • зү№зӮ№пјҡд»»еҠЎеңЁжү§иЎҢиҝҮзЁӢдёӯпјҢеӨ§йғЁеҲҶж—¶й—ҙйғҪеңЁзӯүеҫ… IO ж“ҚдҪңе®ҢжҲҗпјҢCPU еӨ„дәҺз©әй—ІзҠ¶жҖҒгҖӮ
  • й…ҚзҪ®е…¬ејҸпјҡзәҝзЁӢж•° = CPU ж ёеҝғж•° * (1 + зәҝзЁӢзӯүеҫ…ж—¶й—ҙ / зәҝзЁӢ CPU и®Ўз®—ж—¶й—ҙ)
  • з®ҖеҢ–е…¬ејҸпјҡзәҝзЁӢж•° = 2 * CPU ж ёеҝғж•°

2. з»Ҹе…ёе…¬ејҸзҡ„еұҖйҷҗжҖ§дёҺзҺ°д»Ји§ЈеҶіж–№жЎҲвҖӢ

еұҖйҷҗжҖ§пјҡеңЁе®һйҷ…з”ҹдә§зҺҜеўғдёӯпјҢдёҖдёӘеә”з”ЁеҫҖеҫҖж··еҗҲдәҶеӨҡз§Қд»»еҠЎпјҢдё”жөҒйҮҸжҳҜеҠЁжҖҒеҸҳеҢ–зҡ„гҖӮйқҷжҖҒй…ҚзҪ®зҡ„еҸӮж•°еҫҲйҡҫеңЁжүҖжңүеңәжҷҜдёӢйғҪиҫҫеҲ°жңҖдјҳгҖӮ

зҺ°д»Ји§ЈеҶіж–№жЎҲпјҡеҠЁжҖҒеҸҜзӣ‘жҺ§зәҝзЁӢжұ пјҡ

  • ж ёеҝғжҖқжғіпјҡдёҚеҶҷжӯ»зәҝзЁӢжұ еҸӮж•°пјҢиҖҢжҳҜе°Ҷ corePoolSizeгҖҒmaximumPoolSize е’Ң workQueue зҡ„е®№йҮҸзӯүеҸӮж•°жҺҘе…Ҙй…ҚзҪ®дёӯеҝғпјҲеҰӮ NacosгҖҒApolloпјүгҖӮ
  • ThreadPoolExecutor жҸҗдҫӣдәҶеҜ№еә”зҡ„ Setter ж–№жі•пјҢе…Ғи®ёеңЁиҝҗиЎҢж—¶еҠЁжҖҒдҝ®ж”№пјҡ
    • setCorePoolSize(int)
    • setMaximumPoolSize(int)
    • setKeepAliveTime(long, TimeUnit)
  • жіЁж„ҸпјҡLinkedBlockingQueue зҡ„ capacity еӯ—ж®өжҳҜ final дҝ®йҘ°зҡ„пјҢж— жі•зӣҙжҺҘдҝ®ж”№гҖӮеҰӮжһңйңҖиҰҒеҠЁжҖҒдҝ®ж”№йҳҹеҲ—еӨ§е°ҸпјҢйңҖиҰҒиҮӘе®ҡд№үдёҖдёӘеҸҜдҝ®ж”№е®№йҮҸзҡ„йҳҹеҲ—пјҲеҰӮйҮҚеҶҷ LinkedBlockingQueueпјүгҖӮ

еӣӣгҖҒ зәҝзЁӢжұ жәҗз Ғзә§з»ҶиҠӮдёҺеёёи§Ғеқ‘вҖӢ

1. зәҝзЁӢжұ зҠ¶жҖҒжҺ§еҲ¶пјҡctl зҡ„еҰҷз”ЁвҖӢ

ThreadPoolExecutor еҶ…йғЁдҪҝз”ЁдёҖдёӘ AtomicInteger зұ»еһӢзҡ„ ctl еҸҳйҮҸпјҢеҗҢж—¶дҝқеӯҳдәҶзәҝзЁӢжұ иҝҗиЎҢзҠ¶жҖҒпјҲrunStateпјүе’Ңжңүж•ҲзәҝзЁӢж•°пјҲworkerCountпјүпјҡ

  • й«ҳ 3 дҪҚдҝқеӯҳ runStateгҖӮ
  • дҪҺ 29 дҪҚдҝқеӯҳ workerCountгҖӮ
private final AtomicInteger ctl = new AtomicInteger(ctlOf(RUNNING, 0));
private static final int COUNT_BITS = Integer.SIZE - 3;
private static final int COUNT_MASK = (1 << COUNT_BITS) - 1; // 0x1fffffff

// зәҝзЁӢжұ зҡ„ 5 з§ҚзҠ¶жҖҒ
private static final int RUNNING    = -1 << COUNT_BITS; // жҺҘеҸ—ж–°д»»еҠЎе№¶еӨ„зҗҶйҳҹеҲ—д»»еҠЎ
private static final int SHUTDOWN   =  0 << COUNT_BITS; // дёҚжҺҘеҸ—ж–°д»»еҠЎпјҢдҪҶеӨ„зҗҶйҳҹеҲ—д»»еҠЎ
private static final int STOP       =  1 << COUNT_BITS; // дёҚжҺҘеҸ—ж–°д»»еҠЎпјҢдёҚеӨ„зҗҶйҳҹеҲ—д»»еҠЎпјҢе№¶дёӯж–ӯжӯЈеңЁжү§иЎҢзҡ„д»»еҠЎ
private static final int TIDYING    =  2 << COUNT_BITS; // жүҖжңүд»»еҠЎе·Із»ҲжӯўпјҢworkerCount дёә 0пјҢеҚіе°Ҷи°ғз”Ё terminated()
private static final int TERMINATED =  3 << COUNT_BITS; // terminated() жү§иЎҢе®ҢжҜ•

2. йҒҝеқ‘жҢҮеҚ—вҖӢ

OOM йЈҺйҷ©пјҡ

  • й—®йўҳпјҡExecutors.newFixedThreadPool е’Ң newSingleThreadExecutor й»ҳи®ӨдҪҝз”Ёзҡ„ LinkedBlockingQueue жҳҜж— з•Ңзҡ„пјҢе®№жҳ“е Ҷз§Ҝд»»еҠЎеҜјиҮҙ OOMгҖӮExecutors.newCachedThreadPool е…Ғи®ёзҡ„жңҖеӨ§зәҝзЁӢж•°жҳҜ Integer.MAX_VALUEпјҢе®№жҳ“еҲӣе»әеӨ§йҮҸзәҝзЁӢеҜјиҮҙ OOMгҖӮ
  • и§Јжі•пјҡејәеҲ¶дҪҝз”Ё ThreadPoolExecutor жүӢеҠЁеҲӣе»әпјҢжҳҺзЎ®жҢҮе®ҡйҳҹеҲ—еӨ§е°Ҹе’ҢжӢ’з»қзӯ–з•ҘгҖӮ

ејӮеёёдёўеӨұй—®йўҳпјҡ

  • й—®йўҳпјҡдҪҝз”Ё submit(Runnable) жҸҗдәӨд»»еҠЎж—¶пјҢеҰӮжһңд»»еҠЎеҶ…йғЁжҠӣеҮәејӮеёёпјҢиҜҘејӮеёёдјҡиў«е°ҒиЈ…еңЁ Future дёӯпјҢеҰӮжһңдёҚи°ғз”Ё future.get()пјҢејӮеёёдҝЎжҒҜе°Ҷе®Ңе…ЁдёўеӨұгҖӮ
  • и§Јжі•пјҡ
    1. дҪҝз”Ё execute(Runnable) жҸҗдәӨд»»еҠЎпјҢејӮеёёдјҡзӣҙжҺҘжҠӣеҮәеҲ°жҺ§еҲ¶еҸ°жҲ–иў«ж—Ҙеҝ—жҚ•иҺ·гҖӮ
    2. дҪҝз”Ё submit ж—¶пјҢеҝ…йЎ»йҖҡиҝҮ future.get() иҺ·еҸ–иҝ”еӣһеҖје№¶жҚ•иҺ·ејӮеёёгҖӮ
    3. иҮӘе®ҡд№ү ThreadFactoryпјҢи®ҫзҪ® UncaughtExceptionHandlerгҖӮ
