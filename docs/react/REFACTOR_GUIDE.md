### 3. 常见误区与反面教材

每个文档都总结了 4-10 个常见误区，每个误区都包括：
- ❌ 错误的做法与后果
- ✅ 正确的做法
- 💡 解释为什么这样做

**示例**（state-events.md）：

| 误区 | 错误代码 | 正确做法 |
|-----|---------|---------|
| 直接修改 State 对象 | `user.age = 21` | `setUser({ ...user, age: 21 })` |
| 闭包陷阱 | `setCount(count + 1)` 多次只 +1 | `setCount(prev => prev + 1)` |
| setState 后立刻使用新值 | setState 后的下一行拿不到新值 | 使用 useEffect 处理 |
| 直接调用事件处理函数 | `onClick={handleClick()}` | `onClick={handleClick}` |

具体时间因人而异，建议每天投入 2-3 小时学习 + 动手练习。

### Q: 需要有什么前置知识？

**A**: 
- ✅ 基础 JavaScript（变量、函数、数组）
- ✅ HTML 与 CSS 基础
- ❌ 不需要 React 经验

### Q: 可以直接跳到高级阶段吗？

**A**: 
不建议。虽然文档相对独立，但：
- 高级阶段建立在基础概念之上
- 跳过会导致理解不透彻
- 快速开始 + 第一阶段只需 3-4 周

### Q: 练习题的答案唯一吗？

**A**: 
不唯一。参考答案展示的是最佳实践，但：
- 只要满足需求就是正确的
- 鼓励多种实现方式
- 对比参考答案学习最佳实践
