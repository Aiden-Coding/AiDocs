# 机器学习与深度学习基础

本章节从传统机器学习的核心思想讲起，逐步推导到神经网络、多层感知机 (MLP) 与现代 LLM 的基石 Transformer 架构。面向小白拆分为 3 篇教程：

- **[1. 经典机器学习入门](./1-classical-ml-intro.md)**：回归与分类任务、损失函数 (MSE) 求解与过拟合/欠拟合概念。
- **[2. 从零手写多层感知机 (MLP)](./2-neural-network-from-scratch.md)**：激活函数 (ReLU/GELU) 的作用与 PyTorch 神经网络训练全流程。
- **[3. Transformer 与 Self-Attention 机制全解](./3-transformer-and-attention.md)**：自注意力机制 Q/K/V 算例、多头注意力与位置编码组件。
