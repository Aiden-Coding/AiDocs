# 数学与 Python 编程基础

人工智能（特别是机器学习与深度学习）依赖于坚实的数学理论与编程工具链。本章节涵盖 AI 学习所需的核心数学知识与 Python 科学计算栈。

## 核心知识点

### 1. 线性代数 (Linear Algebra)

- **向量与矩阵**：向量加减、点积（Dot Product）、矩阵乘法。
- **特征值与特征向量**：主成分分析（PCA）的数学根基。
- **张量（Tensor）**：高维数组的概念，PyTorch / TensorFlow 的基本数据结构。

### 2. 概率论与数理统计 (Probability & Statistics)

- **常见分布**：高斯分布（正态分布）、伯努利分布、均匀分布。
- **条件概率与贝叶斯定理**：Naive Bayes 分类器与先验/后验概率。
- **期望、方差与协方差**：评估数据离散程度与相关性。

### 3. 微积分与优化 (Calculus & Optimization)

- **偏导数与梯度（Gradient）**：多元函数的导数，损失函数下降的方向。
- **链式法则（Chain Rule）**：深度学习反向传播算法（Backpropagation）的核心。
- **梯度下降法（Gradient Descent）**：SGD、Adam 优化器原理。

### 4. Python AI 工具链

- **NumPy**：高效多维数组运算与矩阵操作。
- **Pandas**：数据清洗、特征提取与表格数据分析。
- **PyTorch**：动态计算图、张量运算与 GPU 加速。
