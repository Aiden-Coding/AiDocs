# 数学与 Python 编程基础

人工智能（特别是机器学习与深度学习）依赖于坚实的数学理论与编程工具链。本章节涵盖 AI 学习所需的核心数学知识与 Python 科学计算栈，面向小白拆分为 3 篇循序渐进的教程：

- **[1. Python 科学计算基础 (NumPy & Pandas)](./1-python-numpy-pandas.md)**：为什么 AI 离不开 NumPy？向量化并行计算与广播机制详解。
- **[2. 直观线性代数与概率论](./2-math-for-ai.md)**：用通俗比喻搞懂 Embedding 向量点积、余弦相似度、矩阵乘法 $Y=XW+b$ 与 Softmax 归一化。
- **[3. PyTorch 张量与自动求导 (Autograd)](./3-pytorch-basics.md)**：GPU 张量计算、动态计算图机制与 20 行代码手写 SGD 参数优化。
