- **[1. Ollama 与本地大模型私有化部署](./1-local-llm-ollama.md)**：大模型界的 Docker，一键部署与兼容 API 调用。
- **[2. vLLM 推理加速与高并发服务](./2-vllm-inference-acceleration.md)**：PagedAttention 虚拟内存管理与高吞吐 HTTP 服务。
- **[3. 模型量化 (GGUF/AWQ) 与服务监控](./3-quantization-and-monitoring.md)**：从 FP16 到 INT4 压缩与 TTFT/TPS 监控指标。
