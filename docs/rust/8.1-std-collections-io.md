# 标准库集合与 I/O

Rust 标准库提供了丰富的集合类型和强大的 I/O 功能。本章介绍常用的数据结构和文件操作。

---

## 标准库类型

### Vec（动态数组）

`Vec<T>` 是可增长的数组，存储在堆上：

```rust
fn main() {
    // 创建 Vec
    let mut v1: Vec<i32> = Vec::new();
    let v2 = vec![1, 2, 3];

    // 添加元素
    v1.push(1);
    v1.push(2);
    v1.push(3);

    // 访问元素
    let third: &i32 = &v2[2];
    println!("The third element is {}", third);

    match v2.get(2) {
        Some(third) => println!("The third element is {}", third),
        None => println!("There is no third element."),
    }

    // 遍历
    for i in &v2 {
        println!("{}", i);
    }

    // 可变遍历
    let mut v3 = vec![1, 2, 3];
    for i in &mut v3 {
        *i += 50;
    }
}
```

### String（字符串）

Rust 有两种字符串类型：

- `String` - 可增长的、可变的、有所有权的 UTF-8 编码字符串
- `&str` - 不可变的字符串切片

```rust
fn main() {
    // 创建 String
    let mut s = String::new();
    let s = "initial contents".to_string();
    let s = String::from("initial contents");

    // 追加字符串
    let mut s = String::from("foo");
    s.push_str("bar");
    s.push('!');

    // 拼接字符串
    let s1 = String::from("Hello, ");
    let s2 = String::from("world!");
    let s3 = s1 + &s2; // s1 被移动，不能再使用

    // 使用 format! 宏
    let s1 = String::from("tic");
    let s2 = String::from("tac");
    let s3 = String::from("toe");
    let s = format!("{}-{}-{}", s1, s2, s3);

    // 遍历字符串
    for c in "नमस्ते".chars() {
        println!("{}", c);
    }

    for b in "नमस्ते".bytes() {
        println!("{}", b);
    }
}
```

### HashMap（哈希表）

```rust
use std::collections::HashMap;

fn main() {
    // 创建 HashMap
    let mut scores = HashMap::new();
    scores.insert(String::from("Blue"), 10);
    scores.insert(String::from("Yellow"), 50);

    // 访问值
    let team_name = String::from("Blue");
    let score = scores.get(&team_name);

    // 遍历
    for (key, value) in &scores {
        println!("{}: {}", key, value);
    }

    // 只在键没有对应值时插入
    scores.entry(String::from("Yellow")).or_insert(50);
    scores.entry(String::from("Blue")).or_insert(50);

    // 根据旧值更新值
    let text = "hello world wonderful world";
    let mut map = HashMap::new();

    for word in text.split_whitespace() {
        let count = map.entry(word).or_insert(0);
        *count += 1;
    }

    println!("{:?}", map);
}
```

### Box（堆分配）

`Box<T>` 是最简单的智能指针，在堆上存储数据：

```rust
    let list = Cons(1, Box::new(Cons(2, Box::new(Cons(3, Box::new(Nil))))));
```

## 文件 I/O

### 读取文件

```rust
use std::fs;
use std::io;

fn main() -> io::Result<()> {
    // 一次性读取整个文件
    let contents = fs::read_to_string("hello.txt")?;
    println!("File contents:\n{}", contents);

    // 读取为字节
    let bytes = fs::read("hello.txt")?;
    println!("File size: {} bytes", bytes.len());

    Ok(())
}
```

### 写入文件

```rust
use std::fs;
use std::io;

fn main() -> io::Result<()> {
    // 写入字符串
    fs::write("output.txt", "Hello, Rust!")?;

    // 写入字节
    fs::write("output.bin", b"Binary data")?;

    Ok(())
}
```

### 使用 File 类型

```rust
use std::fs::File;
use std::io::{self, Read, Write};

fn main() -> io::Result<()> {
    // 打开文件读取
    let mut file = File::open("hello.txt")?;
    let mut contents = String::new();
    file.read_to_string(&mut contents)?;
    println!("{}", contents);

    // 创建并写入文件
    let mut file = File::create("output.txt")?;
    file.write_all(b"Hello, world!")?;

    Ok(())
}
```

### 缓冲读写

使用 `BufReader` 和 `BufWriter` 提高性能：

```rust
use std::fs::File;
use std::io::{self, BufRead, BufReader, BufWriter, Write};

fn main() -> io::Result<()> {
    // 缓冲读取
    let file = File::open("input.txt")?;
    let reader = BufReader::new(file);

    for line in reader.lines() {
        println!("{}", line?);
    }

    // 缓冲写入
    let file = File::create("output.txt")?;
    let mut writer = BufWriter::new(file);

    writeln!(writer, "Line 1")?;
    writeln!(writer, "Line 2")?;
    writeln!(writer, "Line 3")?;

    Ok(())
}
```

## 路径操作

### Path 和 PathBuf

```rust
use std::path::{Path, PathBuf};

fn main() {
    // 创建 Path
    let path = Path::new("./foo/bar.txt");

    // 获取文件名
    if let Some(file_name) = path.file_name() {
        println!("File name: {:?}", file_name);
    }

    // 获取扩展名
    if let Some(ext) = path.extension() {
        println!("Extension: {:?}", ext);
    }

    // 获取父目录
    if let Some(parent) = path.parent() {
        println!("Parent: {:?}", parent);
    }

    // 使用 PathBuf 构建路径
    let mut path_buf = PathBuf::from("/tmp");
    path_buf.push("foo");
    path_buf.push("bar.txt");
    println!("Path: {:?}", path_buf);

    // 设置扩展名
    path_buf.set_extension("md");
    println!("New path: {:?}", path_buf);
}
```

```rust

use std::io;

fn main() -> io::Result<()> {
    // 读取目录内容
    let entries = fs::read_dir(".")?;

    for entry in entries {
        let entry = entry?;
        let path = entry.path();

        if path.is_dir() {
            println!("Directory: {:?}", path);
        } else {
            println!("File: {:?}", path);
        }

    Ok(())
```

### 获取文件元数据

```rust
    println!("Is file: {}", metadata.is_file());
    println!("Read only: {}", metadata.permissions().readonly());

    }

    Ok(())
}
```

```rust
use std::process::{Command, Stdio};
use std::io::Write;

fn main() {
    // 管道示例
    let mut child = Command::new("cat")
        .stdin(Stdio::piped())
        .stdout(Stdio::piped())
```

## 实践示例

### 示例 1：单词计数器

```rust
use std::collections::HashMap;
use std::env;
use std::fs;

fn main() {
    let args: Vec<String> = env::args().collect();
    
    if args.len() < 2 {
        eprintln!("Usage: {} <filename>", args[0]);
        return;
    }

    let contents = fs::read_to_string(&args[1])
        .expect("Failed to read file");

    let mut word_count = HashMap::new();

    for word in contents.split_whitespace() {
        let word = word.to_lowercase();
        *word_count.entry(word).or_insert(0) += 1;
    }

    for (word, count) in word_count.iter() {
        println!("{}: {}", word, count);
    }
}
```

### 示例 2：文件搜索工具

```rust
use std::env;
use std::fs;
use std::io::{self, BufRead, BufReader};

fn main() -> io::Result<()> {
    let args: Vec<String> = env::args().collect();
    
    if args.len() < 3 {
        eprintln!("Usage: {} <pattern> <filename>", args[0]);
        return Ok(());
    }

    let pattern = &args[1];
    let filename = &args[2];

    let file = fs::File::open(filename)?;
    let reader = BufReader::new(file);

    for (line_num, line) in reader.lines().enumerate() {
        let line = line?;
        if line.contains(pattern) {
            println!("{}: {}", line_num + 1, line);
        }
    }

    Ok(())
}
```

## BTreeMap（有序哈希表）

`BTreeMap<K, V>` 按键的自然顺序存储，底层是 B 树。与 `HashMap` 的对比：
- 键必须实现 `Ord`（`HashMap` 只需 `Hash + Eq`）
- 按键有序，支持范围查询
- 内存局部性更好，但增删操作比 `HashMap` 略慢

```rust
use std::collections::BTreeMap;

fn main() {
    let mut scores: BTreeMap<String, u32> = BTreeMap::new();
    scores.insert("Carol".to_string(), 95);
    scores.insert("Alice".to_string(), 88);
    scores.insert("Bob".to_string(), 72);

    // 遍历时按键字典序排列（Alice, Bob, Carol）
    for (name, score) in &scores {
        println!("{}: {}", name, score);
    }

    // 范围查询：取 "Alice"..="Bob" 的所有条目
    use std::ops::Bound::Included;
    for (name, score) in scores.range("Alice".to_string()..="Bob".to_string()) {
        println!("范围内: {} = {}", name, score);
    }

    // first_key_value / last_key_value（最小/最大键）
    if let Some((k, v)) = scores.first_key_value() {
        println!("最小键: {} = {}", k, v);
    }
    if let Some((k, v)) = scores.last_key_value() {
        println!("最大键: {} = {}", k, v);
    }
}
```

## BTreeSet（有序集合）

`BTreeSet<T>` 是 `BTreeMap<T, ()>` 的封装，提供有序、无重复的集合：

```rust
use std::collections::BTreeSet;

fn main() {
    let mut set = BTreeSet::new();
    set.insert(5);
    set.insert(2);
    set.insert(8);
    set.insert(2); // 重复，被忽略

    // 遍历时有序（2, 5, 8）
    println!("{:?}", set); // {2, 5, 8}

    // 范围查询
    let subset: Vec<_> = set.range(3..7).collect();
    println!("3..7 范围: {:?}", subset); // [5]
}
```

## VecDeque（双端队列）

`VecDeque<T>` 是基于环形缓冲区的双端队列，支持在头部和尾部 O(1) 插入和删除：

```rust
use std::collections::VecDeque;

fn main() {
    let mut deque: VecDeque<i32> = VecDeque::new();

    // 尾部追加（类似 Vec::push）
    deque.push_back(1);
    deque.push_back(2);
    deque.push_back(3);

    // 头部插入（Vec 做这个是 O(n)，VecDeque 是 O(1)）
    deque.push_front(0);

    println!("{:?}", deque); // [0, 1, 2, 3]

    // 弹出头部元素（FIFO 队列操作）
    println!("队头: {:?}", deque.pop_front()); // Some(0)
    // 弹出尾部元素（栈操作）
    println!("队尾: {:?}", deque.pop_back());  // Some(3)

    println!("{:?}", deque); // [1, 2]

    // 实用场景：滑动窗口
    let mut window: VecDeque<i32> = VecDeque::with_capacity(3);
    for val in [1, 2, 3, 4, 5] {
        if window.len() == 3 {
            window.pop_front(); // 移出最旧的元素
        }
        window.push_back(val);
        println!("窗口: {:?}", window);
    }
}
```

`VecDeque` 也可通过 `From<Vec<T>>` 与 `Vec` 互转，转换是 O(1)：

```rust
use std::collections::VecDeque;

let vec = vec![1, 2, 3, 4];
let deque: VecDeque<i32> = vec.into(); // O(1)
let back: Vec<i32> = deque.into();     // O(1)
```

## HashSet 集合运算

`HashSet<T>` 支持标准的集合论运算，返回惰性迭代器：

```rust
use std::collections::HashSet;

fn main() {
    let a: HashSet<i32> = [1, 2, 3, 4, 5].into();
    let b: HashSet<i32> = [3, 4, 5, 6, 7].into();

    // 交集：两个集合都有的元素
    let intersection: HashSet<_> = a.intersection(&b).collect();
    println!("交集: {:?}", intersection); // {3, 4, 5}

    // 并集：两个集合的所有元素（去重）
    let union: HashSet<_> = a.union(&b).collect();
    println!("并集: {:?}", union); // {1, 2, 3, 4, 5, 6, 7}

    // 差集：在 a 中但不在 b 中
    let difference: HashSet<_> = a.difference(&b).collect();
    println!("差集 a-b: {:?}", difference); // {1, 2}

    // 对称差集：只在其中一个集合中的元素
    let sym_diff: HashSet<_> = a.symmetric_difference(&b).collect();
    println!("对称差集: {:?}", sym_diff); // {1, 2, 6, 7}

    // 子集/超集检查
    let small: HashSet<i32> = [3, 4].into();
    println!("small ⊆ a: {}", small.is_subset(&a));   // true
    println!("a ⊇ small: {}", a.is_superset(&small)); // true

    // 判断两集合是否不相交
    let c: HashSet<i32> = [10, 11].into();
    println!("a 与 c 不相交: {}", a.is_disjoint(&c)); // true
}
```

## LinkedList（链表）

`LinkedList<T>` 是双向链表。**通常不推荐使用**：在大多数场景下 `Vec` 或 `VecDeque` 性能更好（缓存友好）。仅在需要 O(1) 中间插入/删除且不需要随机访问时考虑：

```rust
use std::collections::LinkedList;

fn main() {
    let mut list: LinkedList<i32> = LinkedList::new();
    list.push_back(1);
    list.push_back(2);
    list.push_front(0);

    println!("{:?}", list); // [0, 1, 2]

    // 拆分链表（O(n) 找到位置后 O(1) 拆分）
    let mut tail = list.split_off(1); // tail = [1, 2], list = [0]
    println!("head: {:?}, tail: {:?}", list, tail);

    // 合并两个链表（O(1)）
    list.append(&mut tail);
    println!("合并后: {:?}", list); // [0, 1, 2]
}
```

> [!TIP]
> 集合选型指南：
> - 需要有序 + 范围查询 → `BTreeMap` / `BTreeSet`
> - 需要头尾高效增删 → `VecDeque`
> - 需要快速查找（不关心顺序）→ `HashMap` / `HashSet`
> - 动态数组（绝大多数场景）→ `Vec`

## 环境变量

```rust
use std::env;

fn main() {

    let exe = env::current_exe().unwrap();
```

## 命令行参数

```rust
use std::env;

fn main() {
    // 收集所有命令行参数（第 0 个是程序名）
    let args: Vec<String> = env::args().collect();
    println!("参数数量: {}", args.len());
    println!("程序名: {}", args[0]);

    // 遍历参数（跳过程序名）
    for (i, arg) in args.iter().enumerate().skip(1) {
        println!("参数[{}]: {}", i, arg);
    }

    // 简单解析：手动处理 --flag 和 --key=value
    let mut verbose = false;
    let mut output = String::from("output.txt");

    let mut iter = args.iter().skip(1);
    while let Some(arg) = iter.next() {
        match arg.as_str() {
            "--verbose" | "-v" => verbose = true,
            "--output" | "-o"  => {
                output = iter.next().cloned().unwrap_or_default();
            }
            s if s.starts_with("--output=") => {
                output = s.trim_start_matches("--output=").to_string();
            }
            other => eprintln!("未知参数: {}", other),
        }
    }

    println!("verbose={}, output={}", verbose, output);
}
```

## 管道与子进程重定向

```rust
use std::process::{Command, Stdio};
use std::io::Write;

fn main() {
    // 捕获子进程输出
    let output = Command::new("echo")
        .arg("Hello from child")
        .output()
        .expect("执行失败");

    println!("stdout: {}", String::from_utf8_lossy(&output.stdout));
    println!("exit code: {}", output.status.code().unwrap_or(-1));

    // 管道：把数据通过 stdin 输入子进程
    let mut child = Command::new("cat")
        .stdin(Stdio::piped())
        .stdout(Stdio::piped())
        .spawn()
        .expect("spawn 失败");

    // 向子进程 stdin 写入数据
    if let Some(mut stdin) = child.stdin.take() {
        stdin.write_all(b"Hello via pipe!\n").unwrap();
    } // stdin 在这里被 drop，子进程收到 EOF

    let output = child.wait_with_output().unwrap();
    println!("cat 输出: {}", String::from_utf8_lossy(&output.stdout));

    // 重定向 stderr
    let output = Command::new("ls")
        .arg("/nonexistent_path")
        .stdout(Stdio::null())   // 丢弃 stdout
        .stderr(Stdio::piped())  // 捕获 stderr
        .output()
        .unwrap();

    println!("错误信息: {}", String::from_utf8_lossy(&output.stderr));

    // 链式管道：ls | grep "Cargo"
    let ls = Command::new("ls")
        .stdout(Stdio::piped())
        .spawn()
        .unwrap();

    let grep = Command::new("grep")
        .arg("Cargo")
        .stdin(ls.stdout.unwrap()) // 把 ls 的输出接到 grep 的输入
        .output()
        .unwrap();

    println!("grep 结果:\n{}", String::from_utf8_lossy(&grep.stdout));
}
```
